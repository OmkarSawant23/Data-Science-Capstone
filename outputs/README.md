# Outputs

This folder contains generated outputs from the IBM Data Science Capstone project.

## Contents

| Output Type | Description |
|---|---|
| Interactive Folium maps | HTML maps showing SpaceX launch sites, success/failure markers, and distance analysis to nearby geographic features. |
| Dashboard screenshots | Screenshots of the Plotly Dash dashboard showing launch site filtering, success pie charts, payload range filtering, and payload-success scatter plots. |
| Visual analysis screenshots | Images used to support findings in the final capstone presentation. |

## Example Files

| File | Description |
|---|---|
| `task1_launch_sites_map.html` | Interactive Folium map showing SpaceX launch site locations. |
| `task2_success_failure_marker_map.html` | Interactive map showing clustered success and failure launch markers. |
| `task3_distance_analysis_map.html` | Interactive map showing distances from launch sites to nearby coastlines, cities, railways, and highways. |
| `dashboard_all_sites.png` | Dashboard screenshot showing all launch sites. |
| `dashboard_selected_site.png` | Dashboard screenshot showing analysis for a selected launch site. |
| `dashboard_payload_scatter.png` | Dashboard screenshot showing payload mass and launch success relationship. |

## Purpose

These outputs provide visual evidence for the analysis completed in the notebooks and dashboard application. They are also used in the final capstone project presentation to communicate findings clearly.
