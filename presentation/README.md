# Presentation

This folder contains the final presentation deliverables for the IBM Data Science Capstone project.

## Contents

| File | Description |
|---|---|
| `Data Science Capstone Project Report.pptx` | Editable PowerPoint presentation summarizing the full SpaceX Falcon 9 landing prediction project. |
| `Data Science Capstone Project Report.pdf` | Final PDF version of the presentation prepared for capstone project submission and grading. |

## Presentation Summary

The presentation summarizes the complete data science workflow completed in this project, including:

- Project overview and executive summary
- Introduction to the SpaceX Falcon 9 landing prediction problem
- Data collection using the SpaceX API
- Web scraping of Falcon 9 launch records
- Data wrangling and feature preparation
- SQL-based exploratory analysis
- Exploratory data visualization using Matplotlib and Seaborn
- Interactive launch site mapping using Folium
- Interactive dashboard development using Plotly Dash
- Machine learning classification models for landing success prediction
- Model comparison and final findings
- Business insights and conclusions

## Purpose

The PDF file is the main submission document for the final capstone assignment. The PowerPoint file is included as the editable source version of the report.
