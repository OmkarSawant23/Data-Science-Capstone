# Notebooks

This folder contains the Jupyter notebooks completed as part of the IBM Data Science Capstone project focused on SpaceX Falcon 9 launch analysis and landing outcome prediction.

## Contents

| Notebook | Description |
|---|---|
| `jupyter-labs-spacex-data-collection-api.ipynb` | Collects SpaceX launch data using the SpaceX REST API, extracts relevant fields, filters Falcon 9 launches, handles missing payload values, and exports the first cleaned dataset. |
| `jupyter-labs-webscraping.ipynb` | Uses web scraping techniques with BeautifulSoup to extract historical Falcon 9 launch records from Wikipedia and prepare them for analysis. |
| `labs-jupyter-spacex-Data wrangling.ipynb` | Performs data wrangling, analyzes launch sites and orbit types, processes mission outcomes, creates the binary landing success class, and exports the cleaned dataset. |
| `jupyter-labs-eda-sql-coursera_sqllite.ipynb` | Uses SQL queries with SQLite to answer key analytical questions about SpaceX launches, payloads, launch sites, booster versions, and landing outcomes. |
| `edadataviz.ipynb` | Performs exploratory data analysis and visualization using Matplotlib and Seaborn. It explores relationships between launch success, payload mass, launch site, orbit type, and flight number. It also prepares one-hot encoded features for machine learning. |
| `lab_jupyter_launch_site_location.ipynb` | Uses Folium to create interactive maps of SpaceX launch sites, visualize successful and failed launches, and calculate distances from launch sites to nearby coastlines, cities, railways, and highways. |
| `SpaceX_Machine Learning Prediction_Part_5.ipynb` | Builds and evaluates machine learning classification models to predict Falcon 9 first-stage landing success using Logistic Regression, SVM, Decision Tree, and K-Nearest Neighbors with GridSearchCV. |

## Purpose

These notebooks document the full data science workflow, including:

- Data collection using APIs
- Web scraping
- Data wrangling and cleaning
- SQL-based analysis
- Exploratory data analysis
- Interactive map visualization
- Feature engineering
- Machine learning prediction

The outputs from these notebooks are used later in the final capstone presentation.Notebook files
